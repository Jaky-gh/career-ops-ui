How to Use Career-Ops

              
The app is built around one loop: fetch, grade, apply, and update history.

            

          


          

            

              
1

              

                
Start on Workflow

                
Workflow is the main page. Use it when you want the app to tell you what to do next in the job search loop.

              

            


            

              
2

              

                
Fetch jobs

                
Click Fetch Jobs to scan configured portals. New matches go into Pipeline as pending jobs that still need grading.

              

            


            

              
3

              

                
Grade the pipeline

                
Click Grade Pipeline to evaluate pending jobs. When grading finishes, click Add to History to merge the generated tracker rows.

              

            


            

              
4

              

                
Apply to strong fits

                
Use Apply-Ready Jobs for roles scoring 4.5 or higher. Open the posting, read the report, submit manually, then click Applied.

              

            


            

              
5

              

                
Use Applications for lookup

                
Applications is the full table. Search and filter when you need to find a company, status, note, or report later.

              

            


            

              
6

              

                
Use Commands for repair

                
Commands is the toolbox. Use Doctor when setup looks wrong, Verify for health checks, and Merge/Dedup/Normalize when tracker data needs cleanup.
